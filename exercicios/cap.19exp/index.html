Não se esqueça, o body também é uma caixa, basta olhar para o plano de fundo deste site.

Personalizando a Aplicação do Background

Para personalizar uma imagem pequena com relação dos eixos x e y basta utilizar os seguintes valores:

background-repeat: repeat; background-repeat: no-repeat; background-repeat: repeat-x; background-repeat: repeat-y;

Também é possível programar quantas vezes deseja que a imagem se repita, e em qual posição da caixa em específico. Utiliza-se os valores:

left top; center top; right top; left center; center center; right center; left bottom; center bottom; right bottom;

Uma outra opção também é redimencionar. Utiliza-se a propriedade: background-size

Os valores aceitos por essa propriedade são: auto; length px/%; cover; contain;.

A última propriedade é o vínculo, ou melhor attachment. Uma das melhores opção quando o conteúdo está vazando para a vertical da página.

Ela aceita os valores: scroll; fixed;

Utilizando Shorthand

Ao invés de utilizar:
background-color: black;
background-image: url();
background-position: center center;
background-repeat: no repeat;
background-attachment: fixed;

Utilize:
..; background: black url() center center no-repeat fixed;

Centralizando Verticalmente um Contêiner

Siga o exemplo...

Para criarmos o contêiner, vamos utilizar os seguintes conceitos em style:

div.outside{
background-color: blue;
height: 40vh;
margin: 200px;
margin-top: 10px;
margin-bottom: 50px;
margin-left: 0%;
}
div.inside{
background-color: blueviolet;
height: 100px;
width: 150px;
}

Vai ficar algo como...

Para centralizar o elemento de dentro do contêiner, vamos adicionar os seguintes conceitos:

div.outside{
background-color: blue;
height: 40vh;
margin: 200px;
margin-top: 10px;
margin-bottom: 50px;
margin-left: 0%;
position: relative;
}
div.inside{
background-color: blueviolet;
height: 100px;
width: 150px;
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
}
